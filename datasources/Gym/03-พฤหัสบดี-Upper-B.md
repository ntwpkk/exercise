# วันพฤหัสบดี — Upper B (เน้นท่าดึง)

---

## วอร์มอัพ (8–10 นาที)

1. **หมุนแขนวงกลม (Arm Circles)** — หมุนไปข้างหน้า 10 รอบ + หมุนไปข้างหลัง 10 รอบ
2. **ดึงยางยืดออกจากกัน (Band Pull-Apart)** — 15 ครั้ง, จับยางยืดระดับหน้าอก ดึงออกจากกัน เน้นบีบสะบักเข้าหากัน (ถ้าไม่มียางยืด ใช้สาย Cable น้ำหนักเบาแทนได้)
3. **หมุนไหล่ออกด้วยสาย Cable น้ำหนักเบา (Cable External Rotation)** — 10 ครั้ง/ข้าง, ข้อศอกชิดลำตัว หมุนมือออกด้านข้าง
4. **Cat-Cow (แอ่น-โก่งหลัง)** — 10 ครั้ง, คุกเข่า สลับโก่งหลัง-แอ่นหลังตามจังหวะหายใจ
5. **ดันสะบักที่กำแพง (Scapular Push-Up แบบ Wall)** — 10 ครั้ง, ยืนหันหน้าเข้ากำแพง มือเท้ากำแพง ดันสะบักออก-หุบเข้า (ใช้กำแพงแทนพื้นเพื่อหลีกเลี่ยงอาการเจ็บข้อศอก)

---

## ท่าที่ 1: Cable Lat Pulldown จับแคบ (ดึงสายเคเบิลฝึกหลัง — มือชิด)

| รายละเอียด | ค่า |
|-----------|-----|
| **กล้ามเนื้อที่ฝึก** | กล้ามเนื้อหลังกว้าง (Latissimus Dorsi), กล้ามเนื้อต้นแขนหน้า (Biceps), กล้ามเนื้อบ่าส่วนล่าง |
| **Sets × Reps** | 3 × 10–12 |
| **พัก** | 90 วินาที |
| **น้ำหนักเริ่มต้น** | 15 kg |

**ทำไมเลือกท่านี้:**
- จับแคบเน้นกล้ามเนื้อหลังกว้างและกล้ามเนื้อใต้สะบัก มากกว่าจับกว้าง
- เป็นท่าหลักของวันเน้นดึง

**วิธีทำ:**
1. ใช้ V-bar หรือ close-grip attachment (มือจับแบบมือชิดกัน)
2. นั่งตรง ขาล็อกใต้แผ่นกัน
3. จับ handle อกยืด ดึงไหล่ลงไม่ให้ยกขึ้น
4. ดึง handle ลงมาที่อกบน ศอกชี้ลงชิดลำตัว
5. บีบสะบักเข้าหากันที่จุดล่างสุด 1 วินาที
6. ปล่อยกลับขึ้นช้าๆ ยืดแขนจนสุด
7. เอนลำตัวน้อยกว่าแบบจับกว้าง

**ข้อควรระวัง:**
- ดึงไหล่ลงไม่ให้ยกขึ้นตลอด — ไม่ยกไหล่
- นึก "ดึงด้วยศอก" ไม่ใช่แขน → กล้ามเนื้อหลังจะทำงานดีขึ้น

## ท่าที่ 2: Dumbbell Row (ดึงดัมเบลฝึกหลัง)

| รายละเอียด | ค่า |
|-----------|-----|
| **กล้ามเนื้อที่ฝึก** | กล้ามเนื้อหลังกว้าง (Lat), กล้ามเนื้อระหว่างสะบัก (Rhomboids), กล้ามเนื้อบ่าส่วนกลาง, ไหล่ด้านหลัง, กล้ามเนื้อต้นแขนหน้า (Biceps) |
| **Sets × Reps** | 3 × 10–12 |
| **พัก** | 90 วินาที |
| **น้ำหนักเริ่มต้น** | 8–10 kg/ข้าง |

**ทำไมเลือกท่านี้:**
- ฝึกการดึงในแนวนอน + บีบสะบักเข้าหากัน → ฝึกหลังให้หนาขึ้น
- สำคัญมากสำหรับคนนั่งโต๊ะที่กล้ามเนื้อระหว่างสะบักมักอ่อนแอ

**วิธีทำ:**
1. มือข้างหนึ่งและเข่าข้างเดียวกันวางบนม้านั่ง (ถ้ามี) หรือยืนเท้าทั้งสองข้าง พับตัวที่สะโพก ทำแบบสองข้างพร้อมกัน
2. มืออีกข้างจับ dumbbell แขนห้อยลงตรง
3. อกขนานพื้น หลังตรง เกร็งท้อง
4. ดึง dumbbell ขึ้นมาข้างลำตัว ศอกชี้ขึ้นฟ้า ดึงไปที่สะโพก
5. บีบสะบักเข้าที่จุดบนสุด 1 วินาที
6. ค่อยๆ ลดลงมา ยืดแขนจนสุด
7. ทำครบแล้วสลับข้าง

**ข้อควรระวัง:**
- อย่าบิดลำตัวขณะดึง — ลำตัวนิ่ง
- อย่าใช้แรงเหวี่ยงดีดน้ำหนักขึ้น

---

## ท่าที่ 3: Cable Seated Row (นั่งดึงสายเคเบิล)

| รายละเอียด | ค่า |
|-----------|-----|
| **กล้ามเนื้อที่ฝึก** | กล้ามเนื้อหลังกว้าง (Lat), กล้ามเนื้อระหว่างสะบัก (Rhomboids), กล้ามเนื้อบ่าส่วนกลาง, ไหล่ด้านหลัง |
| **Sets × Reps** | 3 × 10–12 |
| **พัก** | 90 วินาที |
| **น้ำหนักเริ่มต้น** | 15 kg |

**ทำไมเลือกท่านี้:**
- เพิ่มปริมาณฝึกให้หลังและกล้ามเนื้อระหว่างสะบัก → สำคัญสำหรับคนนั่งโต๊ะที่บ่าส่วนบนตึงแต่สะบักอ่อนแอ

**วิธีทำ:**
1. นั่งที่เครื่อง Cable Row เท้าเหยียบแผ่นรอง เข่างอเล็กน้อย
2. จับ V-bar หรือ bar ตรง (จับแบบฝ่ามือหันเข้าหากันหรือฝ่ามือคว่ำ)
3. อกยืด ดึงไหล่ลง
4. ดึง handle เข้าหาท้องน้อย/สะดือ ศอกชิดลำตัว
5. บีบสะบักเข้าหากันที่จุดสุด 1–2 วินาที
6. ค่อยๆ ปล่อยกลับ ยืดแขนจนรู้สึกยืดที่หลัง
7. ลำตัวเอนไปข้างหน้าเล็กน้อยตอนปล่อย เอนกลับตอนดึง (ไม่โยกมาก)

**ข้อควรระวัง:**
- อย่าโยกตัวมาก — ใช้การบีบสะบักเข้าหากันเป็นหลัก
- เน้นรู้สึก "บีบสะบัก" ทุกครั้ง

---

## ท่าที่ 4: Dumbbell Hammer Curl (ยกดัมเบลงอแขน — ฝ่ามือหันเข้า)

| รายละเอียด | ค่า |
|-----------|-----|
| **กล้ามเนื้อที่ฝึก** | กล้ามเนื้อต้นแขนหน้า (Biceps), กล้ามเนื้อใต้ Biceps (Brachialis — ทำให้แขนดูหนาขึ้น), กล้ามเนื้อแขนท่อนล่าง |
| **Sets × Reps** | 3 × 10–12 |
| **พัก** | 60 วินาที |
| **น้ำหนักเริ่มต้น** | 5 kg/ข้าง |

**ทำไมเลือกท่านี้:**
- จับแบบฝ่ามือหันเข้า (neutral grip) ฝึกกล้ามเนื้อต้นแขนได้ทั้ง 3 มัดพร้อมกัน
- ทำให้ต้นแขนดูหนาขึ้นเมื่อมองด้านข้าง

**วิธีทำ:**
1. ยืนตรง จับ dumbbell ข้างลำตัว ฝ่ามือหันเข้าหากัน
2. ข้อศอกชิดลำตัว นิ่ง
3. งอแขนขึ้นพร้อมกันหรือสลับข้าง จนกล้ามเนื้อต้นแขนหดตัวเต็มที่
4. หยุดสั้นๆ ที่จุดบนสุด 1 วินาที
5. ค่อยๆ ลดลงมาช้าๆ (ช่วงปล่อยน้ำหนักลง 2 วินาที)
6. ฝ่ามือหันเข้าหากันตลอด (ไม่หมุน)

**ข้อควรระวัง:**
- อย่าเหวี่ยงตัว — ถ้าต้องเหวี่ยงแปลว่าหนักเกิน
- ท่อนแขนบนนิ่ง เคลื่อนเฉพาะท่อนแขนล่าง

## ท่าที่ 5: Cable Chest Fly (บินอกด้วยสายเคเบิล)

| รายละเอียด | ค่า |
|-----------|-----|
| **กล้ามเนื้อที่ฝึก** | กล้ามเนื้ออก (Pectoralis Major — เน้นการหุบแขนเข้า), ไหล่ด้านหน้า |
| **Sets × Reps** | 3 × 12–15 |
| **พัก** | 60 วินาที |
| **น้ำหนักเริ่มต้น** | 5 kg/ข้าง |

**ทำไมเลือกท่านี้:**
- ฝึกอกด้วยท่าที่มีภาระข้อศอกน้อย (ข้อศอกไม่ต้องเหยียดรับน้ำหนักหนัก)
- เพิ่มท่าดันเสริมในวันเน้นดึง เพื่อให้สมดุล

**วิธีทำ:**
1. ปรับ Cable ทั้งสองข้างระดับกลาง–สูง
2. จับ handle ทั้งสองข้าง ยืนตรงกลาง ก้าวเท้าหนึ่งไปข้างหน้าเล็กน้อย
3. เอนลำตัวไปข้างหน้านิดหน่อย ข้อศอกงอเล็กน้อย (~10–15°)
4. แขนกางออกจนรู้สึกยืดที่หน้าอก
5. ดึงมือเข้าหากันด้านหน้าอก (เป็นวงโค้ง) จน handle เกือบชนกัน
6. บีบหน้าอก 1–2 วินาทีที่จุดกลาง
7. ค่อยๆ ปล่อยกลับออก ควบคุมช่วงปล่อยน้ำหนักลง

**ข้อควรระวัง:**
- ⚠️ **ข้อศอกซ้าย:** ท่านี้มีภาระข้อศอกน้อย แต่สังเกตอาการ — ถ้าเจ็บให้ลดช่วงการเคลื่อนไหวหรือน้ำหนัก
- อย่าล็อกข้อศอกตรง → งอเล็กน้อยเสมอ

## ท่าที่ 6: Cable Face Pull (ดึงสายเคเบิลเข้าหน้า)

| รายละเอียด | ค่า |
|-----------|-----|
| **กล้ามเนื้อที่ฝึก** | ไหล่ด้านหลัง, กล้ามเนื้อหมุนไหล่ออก (Rotator Cuff), กล้ามเนื้อบ่าส่วนกลาง |
| **Sets × Reps** | 3 × 15–20 |
| **พัก** | 60 วินาที |
| **น้ำหนักเริ่มต้น** | 5 kg |

**ทำไมเลือกท่านี้:**
- ฝึกกล้ามเนื้อหมุนไหล่ออก + ไหล่ด้านหลัง → สำคัญมากสำหรับสุขภาพไหล่ โดยเฉพาะคนนั่งโต๊ะ

**วิธีทำ:**
1. ปรับรอก Cable ระดับหน้า (สูงกว่าศีรษะเล็กน้อย)
2. ใช้เชือก (rope attachment) จับปลายเชือกทั้งสองข้าง
3. ยืนตรง เท้ากว้างเท่าไหล่ ถอยหลังเล็กน้อยให้สายตึง
4. ดึงเชือกมาที่ระดับหน้าผาก–จมูก แยกมือออกจากกัน
5. **หมุนมือออกด้านข้าง** ที่จุดสุด: ให้ท่อนแขนบนตั้งฉากกับลำตัว
6. บีบสะบักเข้าหากัน ค้าง 1–2 วินาที
7. ค่อยๆ ปล่อยกลับ

**ข้อควรระวัง:**
- อย่าใช้น้ำหนักหนักเกิน ท่านี้เน้นการจดจ่อรู้สึกว่ากล้ามเนื้อเป้าหมายทำงาน ไม่ใช่ยกหนัก
- อย่ายกไหล่ขึ้น — เน้นบีบสะบักเข้าและดึงไหล่ลง

## คาร์ดิโอ — วัน Upper (15–20 นาที)

| รายละเอียด | ค่า |
|-----------|-----|
| **อุปกรณ์** | Elliptical (แนะนำ) หรือ ลู่วิ่ง เดิน/วิ่งเหยาะ |
| **ความหนัก** | Zone 2 (โซนคาร์ดิโอระดับเบา — พูดเป็นประโยคได้) — 60–70% ของอัตราหัวใจสูงสุด |
| **อัตราหัวใจเป้าหมาย** | 115–134 bpm |
| **ระยะเวลา** | เริ่ม 15 นาที → ค่อยเพิ่มเป็น 20 นาที |

**วิธีทำ:**
1. เริ่ม 3 นาทีแรก pace เบาๆ (วอร์มอัพ)
2. เพิ่มความเร็ว/แรงต้านจนเข้า Zone 2 — รู้สึกออกแรงเล็กน้อย พูดเป็นประโยคได้
3. รักษาความเร็วคงที่ตลอด
4. 2 นาทีสุดท้าย ลดความเร็วลง (คูลดาวน์)

## ยืดเหยียด — วัน Upper (5–7 นาที)

ค้างท่า stretch 20–30 วินาทีต่อท่า:

1. **ยืดอก (Chest Doorway Stretch)** — ยืนข้างกรอบประตู แขนเท้ากรอบ ศอกระดับไหล่ เอนตัวไปข้างหน้า → ยืดกล้ามเนื้ออกและไหล่ด้านหน้า
2. **ยืดหลังกว้าง (Lat Stretch)** — จับเสา/กรอบประตูข้างเดียว เอนลำตัวออก → ยืดกล้ามเนื้อหลังกว้าง
3. **ยืดต้นแขนหลัง (Tricep Stretch)** — ยกแขนขึ้น งอศอก มือแตะหลัง มืออีกข้างกดศอกเบาๆ → ยืด Triceps (⚠️ ข้อศอกซ้าย: ทำเบาๆ ถ้าเจ็บให้ข้าม)
4. **ยืดคอด้านข้าง (Neck Side Stretch)** — เอียงหัวไปด้านข้าง มือกดเบาๆ ค้าง → ยืดกล้ามเนื้อบ่า
5. **ยืดไหล่หลัง (Cross-body Shoulder Stretch)** — แขนข้ามหน้าอก มืออีกข้างดึงเข้า → ยืดไหล่ด้านหลัง
